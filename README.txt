This application is created using Spring Boot which publishes a RESTful web service based on Spring REST Controller.

The following technologies are used for this implementation:

Core Java
Spring Boot
Spring REST Controller with Jackson API
JSON - For response type

1. Open this project in Eclipse and Import it as an Existing Eclipse Project

2. Run the Application.java

3. It starts in-built Tomcat server on port # 8080. 
Hence, please stop any other service which is running on 8080.

4. Open the browser and visit the following RESTful API:
http://localhost:8080/status
